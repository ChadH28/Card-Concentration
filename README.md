## Project setup
Ensure you have node and npm
```
node -v
```
```
npm -v
```

Ensure you have vue installed globally
```
npm install -g @vue/cli
```
or admin (for linux)
```
sudo npm install -g @vue/cli
```

Run npm i to install all required node modules
```
npm install
```
